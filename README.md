# tracksim.js
An unofficial TrackSim wrapper for NodeJS.

## Installation

You can install this package using Yarn:

```bash
yarn add tracksim.js
```

or NPM:

```bash
npm i tracksim.js
```

## Usage

Wiki is available [here](https://github.com/itzJOHv/tracksim.js/wiki).
